# ikandroid

基于 Jetpack Compose 与 Media3 的原生 Android 影视客户端，数据来自 [ikanbot](https://www1.ikanbot.com/)。

支持搜索与分类、多线路播放、本地收藏、播放历史、剧集自动连播、下一集预缓冲、后台缓存与离线观看，也支持横竖屏、画中画、后台播放和 DLNA 投屏。播放器提供手势调节、轻量倍速与多音轨切换；收藏与播放历史可切换紧凑列表，桌面长按图标还能快速继续最近观看的内容。

海报与标题在列表和详情页之间自然衔接。播放可在详情页、全屏和系统小窗之间连续切换，退出后再次进入同一集也会保留现有播放缓冲。界面支持手机、平板以及跟随壁纸的莫奈配色，并可通过系统分享面板分享 APP。

构建环境：JDK 17、Android SDK 36。

```bash
./gradlew assembleDebug
```

仅供学习交流，请遵守当地法律及数据源站点条款。
